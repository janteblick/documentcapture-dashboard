Privacy Policy

DocumentCapture Dashboard — Chrome Extension — Last updated: 1 April 2026

Overview

DocumentCapture Dashboard is a browser extension that displays pending invoice approval counts and details from Continia Document Capture clusters. This policy describes what data the extension collects, how it is used, and how it is stored.

Data collected and stored

The extension stores the following data locally on your device only, using the browser's built-in chrome.storage.local API:

  • Username and password — entered by the user in the Settings page, used solely to authenticate with the configured Continia Document Capture clusters.
  • Cluster selection — which of the four clusters the user has enabled.
  • Refresh interval and credential expiry — user preferences for how often the dashboard auto-refreshes and how long credentials are remembered.

How data is used

  • Credentials are used exclusively to log in to the documentcapture.eu domains configured by your organisation.
  • No data is sent to any third-party server, analytics service, or external API.
  • No browsing history, personal information, or invoice content is collected or transmitted outside of the documentcapture.eu domains.

Data sharing

The extension does not share, sell, or transmit any user data to any party other than the Continia Document Capture clusters the user explicitly connects to.

Permissions used

  • storage — to save credentials and preferences locally on the device.
  • tabs — to open a temporary background tab on the cluster domain for a same-origin API request; the tab is closed immediately after.
  • scripting — to run a locally-bundled fetch function inside the temporary tab; no external scripts are loaded or executed.
  • host permissions (documentcapture.eu) — to authenticate and retrieve approval data from the clusters the user has access to.

Data retention

All data is stored locally and can be removed at any time by clearing the extension's storage via chrome://extensions → DocumentCapture Dashboard → Clear data, or by uninstalling the extension.

Contact

Questions about this privacy policy can be directed to the person who distributed this extension within your organisation.
