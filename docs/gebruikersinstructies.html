DocumentCapture Dashboard

Chrome Extension — Installation & User Guide

Part 1 — Installation

The extension is currently distributed as a ZIP file while awaiting Chrome Web Store approval. Installation takes about 2 minutes.

Download the extension ZIP file:

Ask the person who sent you this guide for the file, or download it directly:

Download ZIP
  1. Unzip the downloaded file to a permanent location on your PC — for example C:\Extensions\DocumentCapture\
    ⚠ Do not delete this folder after installation — Chrome needs it to stay there.
  2. Open Chrome and navigate to chrome://extensions
  3. Turn on Developer mode using the toggle in the top-right corner of the page.
  4. Click Load unpacked and select the folder you unzipped in step 1.
  5. The DocumentCapture Dashboard extension now appears in the list. Click the puzzle piece icon in the Chrome toolbar (top right) and click the pin icon next to DocumentCapture Dashboard so it stays visible.
After a Chrome update, Developer mode extensions may show a warning asking you to re-enable them. Simply go back to chrome://extensions and click Enable.

Part 2 — First-time setup

  1. Click the DocumentCapture Dashboard icon in the toolbar.
    A new tab opens with the dashboard.
  2. Click Settings (top right of the dashboard).
  3. Clusters — tick only the clusters you have access to:
    • Sneyers
    • Jorssen
    • Gamotors
    • Belien
  4. Enter your Username and Password (same credentials you use to log in to DocumentCapture).
  5. Choose how long credentials should be remembered and how often the dashboard should auto-refresh.
  6. Click Save and close the Settings tab.

Part 3 — Using the dashboard

Click the extension icon in the toolbar to open the dashboard. It shows all your clusters side by side, each with the companies you have access to.

ElementWhat it does
Cluster card One card per cluster (Sneyers, Jorssen, …). Shows the total number of pending invoices for that cluster.
Open ↗ button Opens the cluster's DocumentCapture portal in a new tab.
Company name (blue, underlined) Clickable link — opens the approval page for that specific company directly.
Document column The invoice number. Click it to open the invoice detail page in DocumentCapture.
⤓ icon Downloads the invoice PDF directly.
↺ Refresh button Manually refreshes all clusters immediately.
Auto-refresh The dashboard refreshes automatically at the interval set in Settings (default: every 5 minutes).
✓ No pending approvals All invoices for that company have been approved — nothing left to do.

Part 4 — Troubleshooting

  • Dashboard shows "No credentials configured" — open Settings and save your username and password.
  • "Login failed — check username and password" — verify your credentials are correct in Settings. They are the same as your DocumentCapture login.
  • "No companies found in this cluster" — you may not have access to that cluster, or your account has expired. Untick it in Settings.
  • Extension icon disappeared from toolbar — click the puzzle piece icon and re-pin DocumentCapture Dashboard.
  • Chrome shows a warning after an update — go to chrome://extensions and click Enable next to DocumentCapture Dashboard.
  • Data looks outdated — click ↺ Refresh to force an immediate update.
